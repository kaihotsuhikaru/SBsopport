# SBサポート(SBsopport)

## サイト概要
### サイトテーマ
仕事・私生活の予定を一括管理できるスケジュールサイトです。

### テーマを選んだ理由
自身が抱える問題点でもある計画性やタスクの管理を補う為ひいては同じ悩みを抱える方や日常生活や学業、業務の支えになってほしいと思い選びました。

### ターゲットユーザ
- 学生
- 社会人
- 主婦

### 主な利用シーン
- 日常生活でのスケジュール管理
- 学業や業務でのタスク管理
